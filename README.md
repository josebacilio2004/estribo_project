# ESTRIBO | Arquitectura y Construcción

Professional, modern, and responsive website for the architecture and construction company **ESTRIBO**, based in Huancayo, Perú.

## 🏛️ Project Vision
Inspired by the "Ark Shelter" aesthetic, this website conveys solidity, craftsmanship, and a deep connection to the Andean landscape. The name "Estribo" (Stirrup) symbolizes the essential link between structure and ground.

## 🚀 Features
- **Modern Portfolio**: High-quality imagery showcasing projects in the Mantaro Valley.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile.
- **Direct Connectivity**: Integrated WhatsApp floating button and social media presence.
- **Premium Design**: Vanilla CSS animations, custom SVG icons, and dynamic brand color states.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Grid & Flexbox layouts, custom animations.
- **Vanilla JavaScript**: Intersection Observer for scroll reveals, mobile menu logic.
- **No Frameworks**: Lightweight and high-performance.

## 📁 Repository Structure
- `index.html`: Main landing page.
- `css/styles.css`: Modularized styling.
- `js/main.js`: Interaction logic.
- `assets/`: Organized branding, imagery, and video assets.

## 🌐 Deployment
This site is designed to be hosted on **GitHub Pages**. To activate:
1. Go to **Settings** > **Pages**.
2. Select **Deploy from a branch**.
3. Choose the `main` branch and the `/ (root)` folder.
4. Click **Save**.
